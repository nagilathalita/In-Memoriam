# 🌷 Meu Primeiro Projeto Web

## 🌐 Acesse o site

👉 
https://nagilathalita.github.io/In-Memoriam/

Este foi o meu primeiro projeto desenvolvido com **HTML, CSS e JavaScript**, criado como forma de aprendizado e prática no desenvolvimento web.

Mais do que um simples exercício técnico, este projeto carrega um significado muito especial.

Ele é uma homenagem à minha amada gatinha, que esteve ao meu lado por quase 18 anos. Uma companhia fiel, cheia de amor, que deixou marcas eternas na minha vida.

Este projeto representa não apenas o início da minha jornada na programação, mas também uma forma de eternizar, através da tecnologia, um amor que nunca se apaga.

✨ *“O amor que sentimos nunca morre — ele só muda de lugar.”*

---

## 💻 Tecnologias utilizadas

* HTML
* CSS
* JavaScript

---

## 🎯 Objetivo

Praticar conceitos básicos de desenvolvimento web, como:

* Estruturação de páginas com HTML
* Estilização com CSS
* Interatividade com JavaScript

Cada detalhe foi pensado com carinho, transformando aprendizado em memória, código em sentimento.
